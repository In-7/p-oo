# 巫师3：狂猎

巫师3：狂猎是一款由CD Projekt RED开发的开放世界动作角色扮演游戏。玩家扮演猎魔人杰洛特，在混乱的战争年代中寻找失踪的养女希里。

游戏特色：
- 引人入胜的开放式剧情
- 令人难忘的角色塑造
- 广阔而丰富的游戏世界
- 高度可重玩性
