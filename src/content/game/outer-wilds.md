# Outer Wilds

Outer Wilds是一款由Mobius Digital开发的冒险解谜游戏。玩家扮演一位新晋宇航员，在探索太阳系的过程中发现了一个正在崩塌的时空循环。

游戏特色：
- 独特的22分钟循环时间机制
- 完全开放式的探索
- 没有战斗，只有发现与解谜
- 令人惊叹的宇宙探索体验
