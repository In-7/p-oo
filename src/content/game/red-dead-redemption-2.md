# 荒野大镖客：救赎2

荒野大镖客：救赎2是一款由Rockstar Games开发的开放世界动作冒险游戏。玩家扮演范德林德帮的成员亚瑟·摩根，在美国末代荒野时期体验一段关于救赎的故事。

游戏特色：
- 业界领先的视觉效果
- 深度沉浸的开放世界
- 引人入胜的角色故事
- 丰富多样的游戏活动
