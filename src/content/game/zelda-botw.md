# 塞尔达传说：旷野之息

塞尔达传说：旷野之息是一款由任天堂开发的开放世界动作冒险游戏。玩家扮演林克，在海拉鲁王国中苏醒，面对灾难性的灾厄加侬并拯救公主塞尔达。

游戏特色：
- 开创性的开放世界设计
- 高度自由的游戏方式
- 创新的物理引擎和化学系统
- 令人惊叹的艺术风格
