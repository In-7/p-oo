# 赛博朋克2077

赛博朋克2077 (Cyberpunk 2077) 是 CD Projekt Red 开发的赛博朋克主题开放世界RPG。

## 游戏特色

- **夜之城**：极具未来感的赛博都市
- **深度角色扮演**：选择出身、技能树、升级路线
- **主线剧情**：主线约 20 小时，支线丰富
- **DLSS 支持**：光追效果出色
- **持续更新**：多年优化，完善游戏体验

## 资料片

- **Phantom Liberty**：新增剧情、玩法、地图

## 2024 版更新

- 重制版动画系统
- 画质大幅提升
- 新增内容丰富

## 最低配置

- CPU: Intel Core i7-3770 / AMD Ryzen 5 1600
- 显卡: NVIDIA GTX 780 / AMD RX 470
- 内存: 8GB
- 存储: 70GB SSD
