# Postman

Postman 是 API 开发和测试的终极工具，深受全球开发者喜爱。

## 核心功能

- **API 测试**：发送 HTTP 请求，调试 REST API
- **集合管理**：组织和保存 API 请求
- **环境变量**：多环境配置，快速切换
- **自动化测试**：编写测试脚本，自动验证
- **文档生成**：自动生成 API 文档
- **Mock 服务器**：前端开发无需等待后端

## 适用场景

- 后端 API 开发
- 前后端联调
- 接口自动化测试
- API 文档编写
- 微服务调试
