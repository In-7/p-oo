# Docker

Docker 是最流行的容器化平台，让开发者可以打包应用及其依赖为轻量级容器。

## 核心概念

- **容器**：轻量级、可移植的运行环境
- **镜像**：容器的模板，类似类比
- **仓库**：存储和分发镜像的场所
- **Dockerfile**：定义镜像构建脚本

## 主要优势

- **环境一致性**：开发、测试、生产环境一致
- **快速部署**：秒级启动应用
- **资源隔离**：容器间相互隔离
- **弹性扩展**：轻松扩缩容
- **持续集成**：完美支持 CI/CD

## 适用人群

- 后端开发者
- DevOps 工程师
- 系统管理员
- 全栈开发者
