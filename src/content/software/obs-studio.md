# OBS Studio

OBS Studio (Open Broadcaster Software) 是免费开源的直播和录屏软件，深受全球主播和内容创作者喜爱。

## 核心功能

- **多场景切换**：支持自定义多个场景，一键切换
- **多平台直播**：支持 Twitch、YouTube、Bilibili 等平台
- **屏幕录制**：支持全屏、窗口、特定区域录制
- **音频处理**：支持多音频源混音和滤镜
- **滤镜效果**：内置和第三方滤镜，美化画面
- **高性能编码**：支持多种编码器，画质优秀

## 适用场景

- 游戏直播
- 教学录屏
- 会议录制
- 教程制作
- 远程办公
